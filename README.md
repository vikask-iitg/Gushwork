# Mangalam Hope Pipes Landing Page

A modern single-page product landing page built for Mangalam Hope Pipes. The project is designed to showcase HDPE pipes and fittings with a premium presentation, product highlights, technical specifications, interactive media, and lead-generation forms.

## Overview

This project is a static frontend website built using HTML, CSS, and JavaScript. It focuses on presenting industrial product information in a clean, conversion-oriented layout suitable for a company website, portfolio piece, or product showcase.

## Features

- Product-focused hero section with image gallery
- Interactive thumbnail-based image switching
- Desktop image zoom preview
- Sticky product summary bar on scroll
- Technical specifications section
- FAQ accordion
- Application showcase carousel behavior
- Process step tabs with dynamic content updates
- Testimonial and portfolio sections
- Quote request and catalogue download modals
- Responsive layout for desktop and mobile screens

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

- [index.html](http://_vscodecontentref_/0) — Main page structure
- [styles.css](http://_vscodecontentref_/1) — Complete styling and responsive layout
- [script.js](http://_vscodecontentref_/2) — Interactive functionality for gallery, tabs, accordion, modal, and scroll behavior
- assets/ — Images, icons, and visual resources

## Interactions Included

- Product image gallery with previous and next navigation
- Thumbnail selection for image preview
- Hover zoom effect for product images on larger screens
- Sticky header product summary while scrolling
- FAQ open/close interaction
- Horizontal scrolling support for application and testimonial sections
- Manufacturing process tab switching
- Modal popups for quote requests and brochure downloads
- Escape key and backdrop click support for modal closing

## Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Run the site by opening [index.html](http://_vscodecontentref_/3) in a browser.

You can also use a live server extension in VS Code for a better local development experience.
